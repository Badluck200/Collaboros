# Image Placeholders for Collaboros

Replace the following placeholder images with your own:

## Portfolio Images
- `portfolio-hero.jpg` - Hero section image (recommended: 1200x600px)
- `client-browse.jpg` - Client browsing section image (recommended: 1200x600px)
- `portfolio-1.jpg` - First portfolio example (recommended: 600x400px)
- `portfolio-2.jpg` - Second portfolio example (recommended: 600x400px)
- `portfolio-3.jpg` - Third portfolio example (recommended: 600x400px)
- `portfolio-work-1.jpg` - Portfolio work display (recommended: 400x300px)
- `portfolio-work-2.jpg` - Archived portfolio work display (recommended: 400x300px)

## Profile Images
- `profile-avatar.jpg` - User avatar image (recommended: 200x200px)

## Guidelines
- All images should be optimized for web (use .jpg, .webp, or .png)
- Keep file sizes under 500KB for best performance
- Use consistent aspect ratios as recommended above
- Ensure images represent your brand aesthetic (warm, cozy, professional)

## How to Replace
1. Add your images to this folder (`public/images/`)
2. Use the same filenames as listed above
3. The app will automatically display your images in place of the placeholders
